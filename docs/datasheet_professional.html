Basic Commands

Command Description Example
`scan` Scan for I2C devices and detect EEPROM `scan`
`read ` Read data from memory address `read 0x0000 16`
`write ` Write text to memory address `write 0x0000 Hello World`
`erase` Erase entire EEPROM (fill with 0xFF) `erase`
`wpcheck [addr]` Check write protection status `wpcheck 0x0000`
`addr ` Set EEPROM I2C address manually `addr 0x50`
